Quản lý tài chính cá nhân
Đăng nhập để tiếp tục
Sử dụng tài khoản Google để đăng nhập.
Dữ liệu được mã hoá và chỉ bạn mới xem được.
Kết nối Firebase
Chỉ làm 1 lần duy nhất để lưu trữ và đồng bộ dữ liệu an toàn.
Hướng dẫn tạo Firebase Project
Làm 1 lần duy nhất • Miễn phí hoàn toàn
1 Tạo Firebase Project
Vào console.firebase.google.com → Đăng nhập Google
→ Bấm "Add project" → Đặt tên FinD
→ Tắt Google Analytics (không cần) → Bấm Create project → Chờ ~30 giây
2 Đăng ký Web App & lấy Config
Trong project → Bấm icon </> (Web) → Đặt tên app FinD Web
→ Bấm Register app → Sẽ thấy đoạn code có firebaseConfig
→ Copy các giá trị: apiKey, authDomain, projectId, appId
→ Dán vào 4 ô trong màn hình đăng nhập FinD
3 Bật Google Sign-In
Sidebar trái → Authentication → Get started
→ Tab Sign-in method → Bấm Google → Bật toggle Enable
→ Chọn email hỗ trợ (email của bạn) → Bấm Save
4 Tạo Firestore Database
Sidebar trái → Firestore Database → Create database
→ Chọn Start in production mode → Chọn region asia-southeast1 (Singapore)
→ Bấm Create
5 Cài đặt Security Rules (bảo vệ dữ liệu)
Trong Firestore → tab Rules → Thay toàn bộ bằng code dưới → Publish
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Chỉ user đăng nhập mới đọc/ghi data của chính mình match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
6 Thêm domain vào Authorized domains
Authentication → Settings → Authorized domains
→ Bấm Add domain → Thêm domain GitHub Pages của bạn
(VD: username.github.io) → Add
✅ Sau khi xong 6 bước → quay lại FinD → điền 4 config → Đăng nhập Google → Dữ liệu tự đồng bộ realtime!
